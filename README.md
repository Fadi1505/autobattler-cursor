# Lane Defense 3D — scenes & scripts

This repository intentionally contains **only**:

- `scenes/` — Godot scene files (`.tscn`)
- `scripts/` — GDScript sources (`.gd` and `.uid`)

It is **not** a complete Godot project: there is no `project.godot`, assets, or export config here. To run the game in the editor, copy these folders into a Godot 4.x project that supplies `project.godot`, `assets/`, and autoloads as needed.

For the **Phaser autobattler** (separate project), see: [github.com/Fadi1505/autobattler](https://github.com/Fadi1505/autobattler).
